OSGI compatible Factories for Gosen

As OSGI can not import the same package from multiple bundles it is not possible to add the additional Solr ***Factories provided by Gosen in the org.apache.solr.analysis package. So this provides alternative factories in the org.apache.solr.analysis.gosen package using the Osig** prefix in the class name.

Other than that the implementation provided by this package are the same as of the org.apache.solr.analysis package in the "rel-2.0.2" tag of the Lucene-Gosen repository.
